# Bonjour tout le monde.

## A propos

Ce site à été créer dans le soucis de satisfaire une clientèle désireux souvent de se débarrasser de certains objets de ménage . 
soit parcequ’il ne l’utilise plus, soit parcequ’il est endommagé, soit parcequ’il a été remplacé. 
Bref nous  proposont de les récupérer à un pris jugé pour revendre ceux qui peuvent toujours etres utilisé à un pris abordable pour que tout le monde en sorte gagnant.
Pour vous qui avez un objet quelquonque à déposer, vous pouriez le publier dans la partie Annonce du site.
